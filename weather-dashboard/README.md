# Weather Dashboard 🌦️

A modern and responsive Weather Dashboard built using HTML, CSS, and JavaScript. It provides real-time weather information for any city using the OpenWeatherMap API.

## Features

* Search weather by city name
* Real-time temperature and weather conditions
* Displays:

  * Temperature
  * Feels Like
  * Humidity
  * Wind Speed
  * Pressure
  * Visibility
  * Sunrise & Sunset
* Live date and time display
* Responsive design for all devices
* Stores last searched city using Local Storage
* Error handling and loading states

## Technologies Used

* HTML5
* CSS3
* JavaScript 
* OpenWeatherMap API

## How to Run

1. Download or clone the repository.
2. Add your OpenWeatherMap API key in `script.js`.
3. Open `index.html` using Live Server or any web browser.

## Project Structure

* `index.html` – Page layout
* `style.css` – Styling and responsiveness
* `script.js` – Weather API integration and functionality
